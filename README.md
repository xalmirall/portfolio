# Xavi Almirall | UX/UI Design Portfolio

A personal portfolio website showcasing my work as a UX/UI Designer, including detailed case studies and design processes. 

## 🚀 Live Preview
https://xavialmirall.com

## 📖 About the Project
This repository contains the source code for my professional portfolio. It is designed to be a lightweight, fast, and accessible platform to display my design case studies, highlighting my research, wireframing, and prototyping skills.

## ✨ Features
* **Responsive Design:** Fully fluid layout that adapts to mobile, tablet, and desktop screens.
* **Semantic HTML5:** Built with accessibility and standard web practices in mind.
* **Custom CSS3:** Vanilla CSS implementation for styling and animations without heavy frameworks.
* **Optimized Performance:** Minimal JavaScript for essential interactions (like the scroll-to-top button and responsive navbar).

## 🛠️ Built With
* HTML5
* CSS3
* Vanilla JavaScript
* Google Fonts (Open Sans & Montserrat)
